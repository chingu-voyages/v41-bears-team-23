# App FrontEnd

## Technologies
* Reactjs
* Reduxjs
* Tailwindcss
* Daisy UI
* Vite
* pnpm

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run dev

# production mode
$ pnpm run build
```
